# Image Placeholders

This folder contains lightweight SVG placeholders so the site works out of the box on GitHub Pages.
Replace them with final optimized assets before launch.

## Required replacements
- `hero-poster.svg`, `hero-poster-mobile.svg`: cinematic hero stills (and optional `hero-video.mp4` in `assets/video/`).
- `about-architecture*.svg`: architecture concept render.
- `location-map*.svg`: static map with labels (Svetlana, Dendrary, embankment).
- `masterplan*.svg`: functional zoning / masterplan graphic.
- `materials*.svg`: facade materials and engineering visual.
- `finishing-*.svg`: finishing option visuals.
- `gallery-01..06*.svg`: premium renders (exterior, rooftop, lobby, business spaces, apartment, terrace).
- `og-cover.svg`: social preview image 1200x630.
- `logo-mark.svg`: square brand mark.

## Optimization targets
- Preferred final format: AVIF/WebP (+ JPG fallback where needed).
- Keep hero under ~350 KB per image variant.
- Keep gallery items around ~180-280 KB each.
- Preserve exact aspect ratios already used in HTML/CSS.
