# VOLNA Residence Landing

Статический лендинг для проекта VOLNA Residence.

## Структура

- `index.html` - основная страница
- `404.html` - страница ошибки
- `privacy-policy.html` - политика обработки персональных данных
- `consent-to-processing.html` - согласие на обработку ПД
- `consent-to-ads.html` - согласие на рекламные сообщения
- `robots.txt`, `sitemap.xml` - SEO-служебные файлы
- `assets/css/styles.css` - стили лендинга
- `assets/css/legal.css` - стили юридических страниц
- `assets/js/main.js` - интерактив лендинга
- `assets/js/sendLead.js` - отправка лидов

## Контакт

Основной телефон на сайте: `8 (800) 550-51-20`.

## Юридическая часть

Формы на сайте связаны с:

- Политикой обработки персональных данных
- Согласием на обработку персональных данных
- Согласием на рекламные/информационные рассылки (опционально)

Перед публикацией проверьте и заполните реквизиты оператора ПД на юридических страницах.
